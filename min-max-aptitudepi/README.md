# MinMax

## Due: Mon 11/7 at 11:59 PM

- Create a program called `MinMax.java`
- Using a for loop, prompt the user for 5 numbers from 0 to 99
- After the for loop:
  - Print the smallest number the user enters
  - Print the largest number the user enters

***Example Input:***\
50\
12\
74\
23\
64\
***Example Output:***\
The min is 12\
The max is 74
